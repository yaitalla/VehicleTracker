# VehicleTracker

## Real-time GPS tracker

## Technical specifications
**You must have the node version 10.13.0**

Server side: 
- koa
- socket.io
- xml2js

Client side: 
- create-next-app
- styled-components
- socket.io client

## Get started: 

Clone this project on your machine, open two terminals
```bash
$> cd server
$> npm install
$> npm run dev
```
```bash
$> cd client
$> npm install
$> npm run dev
```
